## ANSI C I/O function

#### open/close function
* fopen         
* fclose        
* fcloseall     
* fflush        

#### read/write function
1. read/write a character
    * fgetc
    * getc
    * fputc
    * putc
2. read/write a line
    * fgets
    * fputs
    * puts
3. read/write a block
    * fread
    * fwrite
    * feof

#### file stream seek function
* ftell
* fseek
* rewind

#### standard input/outpu function
* printf
* scanf
* fprintf
* fscanf
* sprintf
* sscanf
